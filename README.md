# 🎫 Ticket System

A web-based Ticket Management System developed using the MERN Stack. This application allows users to create, track, and manage support tickets efficiently.

## 🚀 Features

- User Registration and Login
- Create New Tickets
- View All Tickets
- Update Ticket Status
- Delete Tickets
- Role-based Access (Admin/User)
- Responsive User Interface

## 🛠️ Technologies Used

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
